Features

📋

ルーティン管理

よく行うトレーニングをテンプレート保存。開始時に選ぶだけで、前回の重量と回数が自動入力されます。

⏱

インターバルタイマー

セット完了と同時にタイマーが自動スタート。30秒〜3分のプリセット対応。終了時バイブ＋通知でお知らせ。

🏆

PR自動更新

種目ごとの最高重量×回数を自動で記録・更新。履歴テーブルで一目で確認できます。

📊

履歴テーブル

種目と日付のクロス集計表で進捗を可視化。記録の編集・削除も簡単に行えます。

☁️

クラウド同期

Googleアカウントでログインすれば、データがクラウドに自動保存。機種変更後もそのまま引き継げます。

📱

ゲストモード

アカウント登録なしで今すぐ使い始められます。あとでGoogleアカウントと連携してデータを引き継ぐことも可能。

プライバシーポリシー

LAST UPDATED: 2026-05-20

はじめに

本プライバシーポリシーは、GymLog（以下「本アプリ」）における個人情報の取り扱いについて説明するものです。本アプリをご利用になることで、本ポリシーに同意したものとみなします。

収集する情報

本アプリは以下の情報を収集・保存します：

  • ワークアウト記録（種目名、重量、回数、実施日時）
  • ルーティン（テンプレート）データ
  • PR（自己最高記録）データ
  • Googleアカウントでログインした場合：メールアドレス、表示名、プロフィール画像URL
  • 匿名ログインの場合：Firebaseが発行する匿名UID
  • アプリ設定（言語設定、タイマーデフォルト秒数）

情報の利用目的

収集した情報は以下の目的でのみ使用します：

  • ワークアウト記録・ルーティン・PRの保存および表示
  • 前回の記録を次のワークアウトに自動入力する機能の提供
  • 複数端末間でのデータ同期（Googleログイン時）
  • アプリの改善および不具合調査

収集した情報を第三者へ販売することは一切ありません。

第三者サービス

本アプリは以下の第三者サービスを利用しています：

  • Firebase (Google LLC)
    ユーザー認証（Firebase Authentication）およびデータベース（Cloud Firestore）の提供に使用します。 Firebase プライバシーポリシー
  • Google Sign-In (Google LLC)
    Googleアカウントでのログイン機能に使用します。 Google プライバシーポリシー
  • Google AdMob (Google LLC)
    アプリ内広告の配信に使用します。AdMobは広告のパーソナライズのために端末識別子等を収集することがあります。 AdMob プライバシーポリシー

データの保存と保持期間

ワークアウトデータはGoogle Cloud（Firebase）のサーバーに暗号化された状態で保存されます。アプリを削除してもクラウド上のデータは保持されます。アプリ設定（言語・タイマー設定）は端末内にのみ保存されます。

データはアカウントを削除するまで保持されます。設定画面の「アカウントとデータを削除」からすべてのデータを削除できます。

ユーザーの権利

本アプリをご利用の方は、以下の権利を有します：

  • 保存されたデータの確認（アプリ内の履歴・ルーティン画面）
  • 個別記録の編集・削除（アプリ内の履歴編集機能）
  • 全データの一括削除（設定 → アカウントとデータを削除）
  • アカウント連携の解除（Googleアカウントの設定から本アプリへのアクセスを取り消し）

子供のプライバシー

本アプリは13歳未満の児童を対象としておらず、意図的に13歳未満の個人情報を収集することはありません。

ポリシーの変更

本プライバシーポリシーは予告なく変更される場合があります。変更後も本アプリをご利用の場合は、変更後のポリシーに同意したものとみなします。最新のポリシーは常に本ページに掲載されます。
